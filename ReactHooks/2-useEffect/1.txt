[ReactHooks]

- useEffect
# 어떤 컴포넌트가 Mount(화면에 렌더링) / Update(다시렌더링, 업데이트) / Unmount(화면에서 사라질때) 됐을때, 
# 특정작업을 실행할 코드를 실행시켜주고 싶을떄 사용한다.

- 사용법
1. 기본적으로 useEffect훅은 인자로 콜백함수를 받는다.
  * 콜백함수란? 다른 함수의 인자로 전달된 함수를 의미한다.

```
# 1. 첫번째 형태 : useEffect의 인자로 하나의 콜백함수만 받는형태
  - 컴포넌트가 매번 렌더링 될때마다 콜백함수가 실행된다.
	- 다시말해서 컴포넌트가 처음 렌더링 될때와, 다시 렌더링 될때 실행된다.

useEffect(() => {
	// 원하는 작업코드
})

# 2. 두번째 형태 : useEffect의 첫번째 인자로 콜백함수, 두번째 인자로 배열(Dependency Array)을 받는 형태 
	- 두번째 형태는 매번 렌더링이 될때마다 실행되는것이 아니라, 
	- 컴포넌트가 맨처음 화면에 렌더링이 될때한번, 배열안의 요소 value값이 바뀔때 실행된다.
	- 만약, 빈배열([])만 있을 경우, 화면에 첫 렌더링이 될때만 실행된다.

useEffect(() => {
	// 원하는 작업코드
}, [value]) // [] 빈배열일때

# 3. Clean up : 더이상 코드가 필요없을때 그 기능을 멈추는 정리작업을 하기위함
	- 혹은, 이벤트리스너를 제거하는 정리작업 처리도 포함

useEffect(() => {
	// 원하는 작업코드의 정리작업을 위해 return을 넣어준다.
	return() => {
		// 해당 컴포넌트가 Unmount될때, 혹은 다음 렌더링시에 불릴 useEffect가 실행되기 이전에 함수가 실행된다.
	}

}, [])


```
